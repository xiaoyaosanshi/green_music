# 快速重启服务说明

## 问题原因

服务仍在运行旧代码，导致：
- 端口 3001 被占用
- nodemon 无法启动新服务
- 代码已更新但服务未生效

## 解决方案

### 方法 1: 使用重启脚本（推荐）

```bash
cd node
# Windows
restart.bat

# 或者手动执行
taskkill /F /IM node.exe
npm run dev
```

### 方法 2: 手动操作

1. **停止所有 Node.js 进程**
   ```bash
   taskkill /F /IM node.exe
   ```

2. **等待 2-3 秒让端口释放**

3. **启动开发服务器**
   ```bash
   cd node
   npm run dev
   ```

### 方法 3: 使用任务管理器

1. 打开任务管理器（Ctrl+Shift+Esc）
2. 找到所有 `node.exe` 进程
3. 右键 → 结束任务
4. 然后在终端运行 `npm run dev`

## 验证服务

服务启动后，测试接口：

```bash
# 健康检查
curl http://localhost:3001/health

# 歌曲列表（应该不再报错）
curl "http://localhost:3001/api/songs?page=1&pageSize=2"
```

## 注意事项

- 确保只有一个 nodemon 进程在运行
- 如果端口仍被占用，检查是否有其他程序在使用 3001 端口
- 修改代码后，nodemon 会自动重启（无需手动操作）
