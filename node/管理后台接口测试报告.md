```bash

# 获取数据统计

  http://localhost:3001/api/admin/statistics

  "http://localhost:3001/api/admin/songs?page=1&pageSize=10"

  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"status":1}'
```

## 修复的问题

### 1. 上传中间件导入错误
- **问题**: `TypeError: upload.fields is not a function`
- **原因**: 导入方式错误
- **修复**: 将 `const upload = require(...)` 改为 `const { upload } = require(...)`

### 2. 管理员登录密码字段错误
- **问题**: `data and hash arguments required`
- **原因**: 数据库字段是 `password_hash`，但代码使用了 `admin.password`
- **修复**: 将 `admin.password` 改为 `admin.password_hash`
