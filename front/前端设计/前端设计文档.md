# Green Music 前端设计文档

## 1. 项目概述

### 1.1 项目简介
Green Music 是一个现代化的音乐播放应用，提供音乐播放、歌单管理、用户交互等功能。

### 1.2 技术栈
- **框架**: Vue 3.x (Composition API)
- **构建工具**: Vite
- **路由**: Vue Router 4.x
- **状态管理**: Pinia
- **UI组件库**: Element Plus / Ant Design Vue (可选)
- **HTTP请求**: Axios
- **样式预处理**: SCSS/SASS
- **代码规范**: ESLint + Prettier
- **类型检查**: TypeScript (推荐)

### 1.3 主题色彩

## 5. API接口封装

### 5.1 Axios配置
```javascript
// api/index.js
import axios from 'axios'
import { useUserStore } from '@/store/user'
import { ElMessage } from 'element-plus'

const service = axios.create({
  baseURL: import.meta.env.VITE_API_BASE_URL,
  timeout: 10000
})

// 请求拦截器
service.interceptors.request.use(
  config => {
    const userStore = useUserStore()
    if (userStore.token) {
      config.headers.Authorization = `Bearer ${userStore.token}`
    }
    return config
  },
  error => Promise.reject(error)
)

// 响应拦截器
service.interceptors.response.use(
  response => {
    const res = response.data
    if (res.code !== 200) {
      ElMessage.error(res.message || '请求失败')
      return Promise.reject(new Error(res.message || '请求失败'))
    }
    return res
  },
  error => {
    ElMessage.error(error.message || '网络错误')
    return Promise.reject(error)
  }
)

export default service
```

## 11. 开发规范

### 11.1 命名规范
- 组件名: PascalCase (如: `MusicPlayer.vue`)
- 文件名: kebab-case (如: `music-player.vue`)
- 变量/函数: camelCase (如: `currentSong`)
- 常量: UPPER_SNAKE_CASE (如: `API_BASE_URL`)

### 11.2 代码规范
- 使用 Composition API
- 单文件组件结构: `<script setup>`, `<template>`, `<style scoped>`
- Props 和 Emits 使用 TypeScript 类型定义（如使用TS）
- 注释规范: 复杂逻辑必须添加注释

### 11.3 Git规范
- 分支命名: `feature/xxx`, `fix/xxx`, `hotfix/xxx`
- 提交信息: `type(scope): message`
  - type: feat, fix, docs, style, refactor, test, chore
